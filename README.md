# Clothing Store Management System

Sistem për menaxhimin e dyqanit të rrobave, i ndërtuar me Node.js, React, Bootstrap dhe MySQL.

## Teknologjitë

Backend:
Node.js, Express.js, Sequelize, MySQL, JWT, Swagger UI

Frontend:
React, Bootstrap, Axios

Database:
MySQL

## Funksionalitetet kryesore

Authentication dhe Authorization me JWT  
Menaxhimi i përdoruesve dhe roleve  
CRUD për produkte, kategori, marka, klientë dhe furnitorë  
Menaxhimi i shitjeve dhe detajeve të shitjeve  
Menaxhimi i porosive të blerjes  
Zbritje dhe oferta  
Karta lojaliteti  
PDF Invoice  
Activity Logs  
Low Stock Alerts  
Dashboard me statistika dhe charts  
Swagger UI për dokumentimin e API-ve
